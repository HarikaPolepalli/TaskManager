# TaskManager

A professional Task Management Application built using HTML, CSS and JavaScript.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks Completed
- Search Tasks
- Filter Tasks
- Priority Levels
- Categories
- Due Dates
- Dark Mode
- Progress Bar
- Local Storage

## Technologies Used

- HTML
- CSS
- JavaScript
